# Contradictory, My Dear Watson

Loose collection of notebooks for the Kaggle competition. I validated on a 1.2k split from the train set (competition uses TPUs, so no live LB numbers here).

## Notebooks
- `bert.ipynb` – fine-tune BERT; ~0.667 acc.
- `qwen_icl_0_5B.ipynb` – Qwen 3 0.5B ICL; ~0.552 acc.
- `qwen_icl_1.7B.ipynb` – Qwen 3 1.7B ICL; ~0.760 acc.
- `qwen_icl_4B.ipynb` – Qwen 3 4B ICL; ~0.785 acc.
- `qwen_sft_watson.ipynb` – LoRA SFT on Qwen 1.7B; ~0.850 acc.
